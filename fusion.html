Nanoflux Fusion is an evolved/extended version of Nanoflux. It is built on top of it, and entirely compatible. Like NanoFlux, Fusion is a completely dependency-less and micro-sized implementation. Inspired by Redux, it uses functions to define the application state. This way, managing the application state is even more comfortable. Furthermore, Nanoflux Fusion guarantees immutable application states, making it still more robust and free of undesired side-effects.
