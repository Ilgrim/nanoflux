--- layout: null section-type: about title: About --- ## About nanoflux is a very tiny and self-containing implementation (3.5 KiB minified, 1.25 KiB gzipped) of the Flux architecture. It offers a comfortable API, reducing boilerplate code. It has absolutely no dependencies and can be used in any component based context, e.g. ReactJS, Angular, Polymer, RiotJS etc. Support the repo by starring or forking it!
